# 宠物品种颜色特征大全

> 涵盖猫、狗、小型哺乳动物、鸟类、爬行动物、鱼类  
> 每品种列出毛色 / 体色 / 羽色与瞳色或其他颜色特征

---

## 🐱 一、猫咪（Cat）

| 品种 | 毛色 / 体色 | 瞳色 / 其他颜色特征 |
|------|------------|------------------|
| 英国短毛猫 | 蓝灰、白、黑、金渐层、银渐层、虎斑、双色等 | 铜色、金色、蓝色（蓝色毛常见） |
| 美国短毛猫 | 银色虎斑、红色虎斑、黑、白、双色等 | 金色、绿色、蓝色 |
| 波斯猫 | 白、黑、蓝、奶油、红、银渐层、金渐层、玳瑁等 | 蓝色（白色/浅色）、铜色、绿色 |
| 布偶猫 | 重点色（海豹/蓝/巧克力/丁香）配白色，双色、手套等 | 蓝色（标准） |
| 苏格兰折耳猫 | 蓝、白、奶油、虎斑、玳瑁、双色等 | 金色、铜色（配合毛色） |
| 缅因猫 | 棕虎斑最常见，白、黑、蓝、红、奶油及多种虎斑 | 金色、绿色、铜色 |
| 暹罗猫 | 重点色（海豹、蓝、巧克力、丁香）配浅米色身体 | 蓝色（品种特征） |
| 孟加拉猫 | 棕豹纹、银豹纹、蓝豹纹、雪色豹纹 | 金色、绿色、蓝色 |
| 挪威森林猫 | 几乎所有颜色，虎斑最多见 | 金色、绿色、铜色 |
| 阿比西尼亚猫 | 黄褐色（Ruddy）、红、蓝、淡黄褐，毛尖分色 | 金色、绿色、琥珀色 |
| 俄罗斯蓝猫 | 蓝灰色（唯一标准色） | 翠绿色（品种特征） |
| 无毛猫（斯芬克斯） | 皮肤色（桃色、黑、蓝、虎斑纹路等） | 多色，无特定限制 |
| 波米拉猫 | 银色、金色为主，毛尖染色 | 绿色（品种特征） |
| 东方短毛猫 | 白、黑、巧克力、蓝、丁香，共 300+ 颜色组合 | 绿色（多数）；白色为蓝色或鸳鸯眼 |
| 缅甸猫 | 貂褐、蓝、巧克力、丁香、红、奶油、玳瑁等 | 金色、黄色 |
| **德文卷毛猫（Devon Rex）** | **白、黑、蓝、奶油、红、银渐层、玳瑁、虎斑、双色等，几乎所有颜色** | **金色、绿色、蓝色、鸳鸯眼；白色个体常见蓝眼** |

> ※ 重点色猫（暹罗、布偶等）的颜色会随温度和年龄变化；白色猫若为蓝眼，耳聋风险较高。  
> ※ 德文卷毛猫因毛发卷曲、毛量稀少，颜色看起来比实际更浅，选色时需注意。

## 🐶 二、犬类（Dog）

| 品种 | 毛色 / 体色 | 瞳色 / 其他颜色特征 |
|------|------------|------------------|
| 金毛寻回犬 | 浅奶油、金黄、深金至红棕 | 棕色、深棕色 |
| 拉布拉多寻回犬 | 黄色、黑色、巧克力色 | 棕色、深棕（黑毛）、榛色 |
| 德国牧羊犬 | 黑背棕腿（鞍形）、黑、白、双色 | 棕色（主流） |
| 法国斗牛犬 | 奶油、浅黄褐、虎斑、白、虎斑配白 | 深棕色 |
| 柯基犬（彭布罗克） | 红白、貂色白、黑白三色 | 棕色、深棕色 |
| 边境牧羊犬 | 黑白最常见，另有红白、蓝白、大理石纹 | 棕色；大理石纹可有蓝色 |
| 哈士奇 | 黑白、灰白、红白、纯白 | 蓝色、棕色或鸳鸯眼（品种特征） |
| 萨摩耶犬 | 纯白、白奶油色 | 深棕色、黑色 |
| 贵宾犬（标准/迷你/玩具） | 白、黑、杏色、棕、银、红、蓝灰等 | 深棕色（深色毛）；琥珀色（浅色毛） |
| 柴犬 | 红柴、黑芝麻、黑褐三色、奶油（裏白） | 深棕色 |
| 比格犬 | 三色（黑棕白）、柠檬白双色、红白双色 | 棕色、榛色 |
| 马尔济斯犬 | 纯白色 | 深棕色或黑色 |
| 吉娃娃 | 几乎所有颜色及组合，短毛/长毛 | 棕色、深棕色 |
| 博美犬 | 橙色、红色、白、黑、奶油、棕、虎斑等 | 深棕色、黑色 |
| 雪纳瑞（迷你） | 椒盐色、黑银、纯黑 | 深棕色 |
| 阿拉斯加雪橇犬 | 黑白、灰白、红白、貂色 | 棕色（不可为蓝色，品种规定） |
| 约克夏梗 | 钢蓝背配棕色头腿 | 深棕色、黑色 |
| 比熊犬 | 纯白（偶见杏色或奶油轮廓） | 深棕色或黑色 |
| 秋田犬 | 红色、白色、虎斑，配白色裏白 | 深棕色 |
| 杜宾犬 | 黑棕、红棕、蓝棕、白化（罕见） | 深棕色；蓝色毛可能有蓝眼 |

> ※ 部分犬种（如哈士奇、边牧）可出现鸳鸯眼（一蓝一棕）；大麦町犬虽未列，其黑斑/肝斑配白色最具代表性。

## 🐹 三、小型哺乳动物

| 品种 | 毛色 / 体色 | 瞳色 / 其他颜色特征 |
|------|------------|------------------|
| 荷兰侏儒兔 | 白、黑、蓝、灰、棕、貂色等多种 | 棕色、蓝色（白化为红眼） |
| 垂耳兔（荷兰/美国） | 白、灰、棕、黑、虎斑等 | 棕色、蓝色（白化为红眼） |
| 安哥拉兔 | 白、黑、巧克力、蓝、浅黄褐等长毛 | 棕色（红眼为白化） |
| 金丝熊（叙利亚） | 金棕（野生色）、白、奶油、肉桂、玳瑁等 | 黑色（深色毛）；红色（白化） |
| 侏儒仓鼠 | 灰白（野生）、珍珠白、蓝宝石等 | 黑色 |
| 豚鼠（荷兰猪） | 白、黑、棕、奶油、玳瑁等多种 | 棕色、红色（白化） |
| 费雷特（雪貂） | 白貂（白）、黑貂、银貂、混色雪貂（Panda） | 黑色、红色（白化）、蓝色（稀有） |
| 龙猫 | 标准灰、白、米白、紫罗兰、黑天鹅绒等 | 黑色、红色（白化品系） |

> ※ 白化品系（Albino）动物眼睛因缺乏黑色素，瞳孔会呈红色（实为血管颜色透出）。

## 🦜 四、观赏鸟（Bird）

| 品种 | 羽色 | 眼睛 / 其他颜色特征 |
|------|------|------------------|
| 虎皮鹦鹉 | 绿（野生）、蓝、黄、白、灰、紫等，超百种色系 | 黑色瞳孔、彩色虹膜（因品系） |
| 玄凤鹦鹉 | 正常灰、黄化（Lutino）、白化、珍珠、肉桂等 | 黑色（深色），红色（黄化/白化） |
| 太阳锥尾鹦鹉 | 橙黄红渐变胸腹，绿翼，蓝尾 | 深棕/黑色 |
| 金刚鹦鹉 | 蓝黄、猩红、绿翼、紫蓝等，颜色鲜艳 | 淡黄色、白色 |
| 非洲灰鹦鹉 | 深灰体，红色尾羽（刚果）/暗红（提姆内） | 淡黄色 |
| 牡丹鹦鹉（爱情鸟） | 绿、蓝、黄、白、绿黄多种，头部常有对比色 | 深棕或黑色 |
| 文鸟（白文鸟/十姐妹） | 白、杂色、棕、桂皮色 | 黑色 |
| 芙蓉鸟（金丝雀） | 黄、橙、红、白、绿色系 | 黑色 |
| 八哥（一般/大八哥） | 黑色羽毛，翼有白斑 | 橙黄色 |

> ※ 鸟类颜色特征除眼睛外，还常包括：嘴喙颜色、蜡膜颜色（虎皮鹦鹉）、脸颊斑点色等。

## 🦎 五、爬行动物（Reptile）

| 品种 | 体色 / 变种 | 眼睛 / 其他颜色特征 |
|------|------------|------------------|
| 豹纹守宫 | 野生黄褐黑斑、白雪、暴风雪、星系、丁克等多色变种 | 黑色竖瞳 |
| 胡须龙（鬃狮蜥） | 棕、橙、黄、红，人工选育有橙红、白化 | 深棕色（可因温度变色） |
| 球蟒 | 正常色（棕黄黑）、白化、蜘蛛、幽灵等超 100 种变种 | 黑色（正常）；红色（白化） |
| 玉米蛇 | 正常橙红、奶油、雪、珊瑚、血红等多色变种 | 圆形瞳孔，棕色 |
| 绿鬣蜥 | 绿色为主，可变为橙（发情期）、棕灰 | 金色、黄色 |

> ※ 爬行动物颜色特征还包括：脱皮前体色变暗、情绪激动时皮肤显色变化（变色龙、变色石龙子等）。

## 🐠 六、观赏鱼（Fish）

| 品种 | 体色 / 鳍色 | 观赏重点 |
|------|------------|---------|
| 孔雀鱼 | 多色，雄鱼尾色鲜艳（红、蓝、黄、虹彩等无限组合） | 体色及尾鳍图案 |
| 斗鱼（泰国斗鱼） | 红、蓝、绿、紫、白、多色渐变，超两百种 | 鳍型和体色 |
| 金鱼 | 红、橙、黄、白、黑、蓝灰、花斑等 | 体型和颜色 |
| 锦鲤 | 红白（红白锦鲤）、黑红（昭和三色）、黄金、松叶等 | 花纹分布与对称性 |
| 神仙鱼（燕鱼） | 银黑条纹（野生）、金色、白、大理石、黑、蕾丝等 | 体色与鳍型 |
| 荧光灯鱼（红绿灯） | 蓝绿荧光线配红色腹部 | 荧光线色彩鲜艳度 |

> ※ 鱼类的"颜色特征"主要体现在体色、鳍型和花纹图案；部分品种如孔雀鱼雌雄体色差异显著。

---

*说明：以上颜色特征以常见家养宠物品种为主，部分品种（如猫、狗）在人工育种下颜色变种极多，仅列举代表性颜色。具体配色标准以各国猫/犬协会（CFA、TICA、AKC 等）官方血统书为准。*
